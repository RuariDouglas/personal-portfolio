React Guardian App

Live demo See the Code

Using The Guardian’s API & Style guide with The Open Weather API, I designed and built a fully responsive, lite version of the current Guardian site. Built with React (w/Redux) & Firebase, this CRUD app fetches daily news articles, allows users to sign in, create/edit/delete comments & search the guardian. For the design, my goal was to modernising the aesthetic whilst ahering to brand guidelines.
