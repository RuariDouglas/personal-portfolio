React Guardian App

See Live View Code

A fully responsive react CRUD application using The Guardian News & Open Weather Map API with Firebase for Database, Auth and Hosting.

This app features:
- Various React hooks such as useEffect, useHistory and useLocation to make API calls and define routes.
- Multiple API calls to fetch News and Weather data.
- State stored and persisted using Redux and local storage.
- User authentication to allow for article comment creation, deletion and editing.

I worked with The Guardian's style guide to retain brand consitency. MaterializeUI was used for the icons.

Shift Tracker

See Live View Code

A fully responsive back-end focussed CRUD project built with MongoDB, Express & Node. The application requires you to enter your work shift details, in turn through Javascript; exact hours are calculated and stored in the database and a total hours figure is displayed for each month.

This app features:
- Form validation through Javascript.
- CRUD routing with Express router.
- User authentication using passport local mongoose and MongoDb.

I built this for my partner who was spending too much time calculating her hours manually.
